This project is a simple Java Spring Boot RESTful API used to manage a collection of items.
Items are stored in memory using an ArrayList.

Technologies Used:
  -Java
  -Spring Boot
  -Maven
  
How to run the Application:
  -Import the project into Eclipse or IntelliJ.
  -Run DemoApplication.java.
  -The server will start on:http://localhost:8081
  
Add end points:
 -Add new item
  -POST /items
 -Get Item by id 
  -GET /items/{id}
